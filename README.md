# Sistema de Gestión de Solicitudes de Ejemplares
Este proyecto es una aplicación web desarrollada con **Java** y **Spring Boot** para la Práctica III de Taller Transversal I.

## Objetivos
* Implementar lógica de negocio siguiendo la metodología **TDD** (Test Driven Development).
* Gestionar dependencias mediante **Maven**.
* Realizar pruebas unitarias con **JUnit**.

## Cómo ejecutar
1. Compilar: `mvn clean install`
2. Ejecutar: `mvn spring-boot:run`
